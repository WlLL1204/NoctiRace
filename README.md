RimworldのHAR種族MOD【Nocti Race】の開発リポジトリ

夜に生きる吸血蝙蝠であるノクティを追加します。

# 【ノクティとは】
蝙蝠の遺伝子を持ちつものの身体能力自体は人間と大差はありません、しかし彼女たちの生態はヒトとは大きく異なります。
ノクティは摂取した血液を保存し消費することで特別な能力を行使したり、エコロケーションと暗視の能力によって暗い環境でも問題なく活動ができます。
人間のことが大好きなため、コロニー内に人間がいると少し幸せになります、囚人として捕まえているともっと良いです。
吸血蝙蝠であるため本能的に人間の血肉を求めています、薬物依存やアルコール中毒のようにヘモゲンを定期的に摂取しないと不機嫌になり、人肉を食べても不幸になりません。~~むしろ少し喜びます~~
ノクティにとって人間は、良き友人であり、大切な仲間であり、愛すべき恋人であり、最高級の食材なのです。

# 【種族的な特性】
・全身が毛で覆われているため寒さに強い<br/>
・眠った時の寒さへのデバフがない、同じベッドで寝ている人もなくなる<br/>
・夜行性で夜に起きていると有利<br/>
・ヘモゲンを食料として美味しく食べられる<br/>
・空を飛べるから障害物による減速や、キャラバン編成時の地形へのデバフがほとんどない<br/>
・全身毛だらけだから裸でも気にしない<br/>

# 【夜行性について】
・夜間に起きていると心情アップ(よたか見たいなもの)<br/>
・暗闇でのあらゆるデバフ無し(遺伝子の暗視と同じ)<br/>
・月光を浴びているとサイフォーカスが溜まっていく<br/>
・意識のない生き物を極めて少ない外傷で即死させれる！狩猟に便利<br/>

# 【夜間で使える設備】
・月光で発電できる太陽光発電機<br/>
・夜間に育つジャガイモとかお米<br/>

# 【血液の技術】
捕まえた人間の近くにバイオフェライトハーベスターのような機会を置いて自動で安全に血液を搾取しよう！
搾取した血液はDubs bad hygineの水道管パイプラインのようにタンクに貯めたり、飲み血液として頂いたりできる！
血液は万能なエネルギー！パイプラインを繋げればノクティ独自のタレットや生産設備を稼働させられるぞ！
が、搾取用の囚人が増えてきたらコロニーが手狭になってしまう！それを解決するために人間を可逆圧縮することが可能！
沢山の人間をコンパクトにまとめて大規模な血液パイプラインを構築できる。が、ノクティは可逆圧縮された人間が好きじゃないから食用には使えなくなる

# 【生け捕りの技術】
ノクティたちは人間を生け捕りにするために様々な武器や技術を開発してきました<br/>
・外傷を抑え血液だけを抽出することで意識不明にする銃<br/>
・血液を自動で抜き続けて意識不明を維持する搾血機<br/>
・怪我を素早く塞ぎ、同時に意識を無くす特殊な治療法<br/>
など
